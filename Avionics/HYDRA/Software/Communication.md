# Communication
This document covers the communication of HYDRA 

## Requirments 
- Altitude 
- Pitch, Yaw, Roll
- Velocities 
- Accelerations
- States 
- Power status
- Radio status 

## Rocket to Ground
Using an RFD 900 to transmit data on 902MHz.
Current packet system is COBS, to be changed to MAVLINK.
